Functions Used

checkPassword()

This function is run each time the text is updated in the input box in the password checker. It runs the other functions and changes the colours of the cells based on those results.

checkForNumber()

This is used to check whether the password contains a number using the isNaN() function

checkForSymbol(), checkForUpperAndLower()

These two functions both use regular expressions to check whether the password meets their requirements

checkForCommonPasswords(), checkForWords()

These two functions search through an array to see whether the password is in it or contains that word. The arrays were created by obtaining a word list from the internet and then them being formatted into an array using a python script created by myself.
